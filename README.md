# Customer Churn Prediction – Data Preprocessing & Feature Engineering

---

## 📌 Project Overview

This project focuses on building a complete **data preprocessing and feature engineering pipeline** for predicting customer churn. Customer churn refers to customers discontinuing a service or subscription. Predicting churn allows businesses to proactively identify at-risk customers and implement retention strategies.

The project demonstrates a full **machine learning data preparation workflow**, transforming raw customer data into a structured dataset suitable for predictive modeling. The workflow includes handling categorical variables, feature scaling, outlier detection, feature engineering, feature selection, and the creation of a complete preprocessing pipeline.

The final output includes a machine learning pipeline capable of preprocessing data and predicting whether a customer is likely to churn.

---

## 🎯 Project Objectives

* Build a complete data preprocessing pipeline for machine learning
* Convert categorical variables into numerical formats
* Apply feature scaling techniques to normalize numerical data
* Detect and handle outliers in customer data
* Engineer new features to capture customer behavior patterns
* Select the most relevant features for churn prediction
* Implement an end-to-end machine learning pipeline
* Train and evaluate a churn prediction model
